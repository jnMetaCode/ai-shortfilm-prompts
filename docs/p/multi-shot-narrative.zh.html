多分镜叙事短片模板

从《丧尸清道夫》抽出来的多镜头叙事骨架。 适合做剪辑型短片：一个事件分成 N 个分镜，每个 5-8 秒，最后剪辑成连贯故事。

骨架结构

【基础设定】
  人物1（参考 Portrait1）：…
  人物2（参考 Portrait2）：…
  场景（参考 Portrait3）：…
  声音：…

【氛围与画质】
  风格核心：…
  视觉基调：…
  色彩与影调：…

【画面内容】
  分镜一：景别 + 构图 + 运镜手法 + 画面内容
  分镜二：…
  分镜三：…
  …

每个分镜四件套：景别 / 构图 / 运镜手法 / 画面内容。

分镜词汇表

挑词的时候照这个查：

景别

  • 大全景：交代环境
  • 全景：人物 + 一定的环境
  • 中景：腰部以上
  • 近景：胸部以上
  • 特写：脸、手等局部
  • 长焦特写：远距离压缩透视的特写

构图

  • 居中构图 / 对称构图
  • 三分线构图 / 九宫格
  • 对角线构图
  • 框景构图（用门窗框柱视觉主体）
  • 前景 / 中景 / 背景（明确每一层是什么）
  • 空镜（画面里没有人物主体）
  • 过肩镜头（前景是一个人物的肩 / 后脑）

运镜手法

  • 固定机位拍摄
  • 镜头缓慢前推 / 后拉
  • 镜头跟随运动拍摄
  • 镜头环绕拍摄（顺时针 / 逆时针）
  • 手持拍摄（带轻微浮动）
  • 平移镜头 / 摇镜头
  • FPV 模式（画面自由倾斜）

画面内容（叙事用动作动词）

  • 进入 / 转身 / 后撤 / 站直 / 下蹲
  • 低头 / 抬头 / 歪头 / 注视
  • 切换表情 / 防御姿态 / 准备战斗

套用示例（机器人 + 小动物 + 度假别墅 类型）

几个高分镜手法

1. 空镜 + 画外音

分镜：景别全景，平视拍摄吧台全貌。空镜，画面下方三分之一处为吧台。
画面内容：吧台后方传来悉悉簌簌的声音，像是有什么东西躲在吧台下翻找。

用法：用一个没有主角的镜头 + 画外声效，建立"未知威胁"感。

2. 过肩特写（建立两人的注视关系）

分镜：长焦，过肩镜头。
构图：机器人作为远景处于画面左侧，前景为鸵鸟后脑位置画面右侧。
     焦点对焦于机器人，产生浅景深的透视关系。
运镜手法：固定机位拍摄。
画面内容：鸵鸟后脑勺从画面下方入画窜出，机器人被吓到，双手抱头。

用法：两个角色对峙时，用过肩镜头一次拍到两个人，比来回切镜头紧张感强。

3. 表情切换作为镜头终点

分镜：长焦特写，拍摄机器人侧面 45 度面部表情。
画面内容：机器人面部表情由「黄色害怕」切换至「白色思考」。

用法：一个完整分镜可以只表现"情绪变化"。表情切换 = 思考过程，是没有台词时的内心独白。

4. 跳切 + 跟拍 + 场景转换（适合 MV 段）

画面内容：机器人在场景各处（泳池边、二楼走廊、躺椅上、房顶）独舞。
跟拍运镜，根据画面自动构图，跳切分镜，转换场景，
穿插脚部（后滑太空步）特写。

用法：MV 不需要每个镜头都精心设计，给 AI 一个跳切自由就行，关键是动作风格定准（"完美复刻迈克尔·杰克逊"）。

调试建议

  1. 先做 2-3 个分镜试一下质感，别一上来铺 7 个分镜。
  2. 分镜之间的运动方向要呼应（前一个镜头主角往右走，下一个镜头主角应该从左入画）。
  3. 每个分镜独立生成 5-10 秒，剪辑里靠转场拼接。一次性 30 秒一镜到底成功率极低。
  4. Portrait 编号要稳定：Portrait1 永远是主角，Portrait2 永远是动物，不要中途换编号。

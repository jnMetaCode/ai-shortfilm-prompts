视频提示词

武器充能（做 10s 或者 15s 都可以）

视觉基调

  • 画面风格：赛博朋克、硬核科幻、电影级质感、超写实、真人实景拍摄、废土工业风。
  • 色彩与色调：好莱坞青橙色调。冷峻的橙灰色和暗青色，营造出压抑、冰冷和未知的氛围。高亮度的霓虹橙色和青蓝色。极端的"冷暖对比"为核心视觉特征。
  • 光影设计：暗调低照明、高对比度。暗部信息压缩，保留细节。边缘添加轻微柔焦与适度的胶片颗粒感。画面整体可见度极低。
  • 环境光：浓密的体积雾漫射出的微弱冷光。
  • 光源：发光道具作为动态光源（头盔玻璃的红色 HUD、机械装甲的橙色灯光、刀刃的橙色强光、怪物的青蓝色眼睛），产生强烈的边缘光和镜头光晕。
  • 拍摄设备：索尼威尼斯电影机拍摄，搭配佳能 K-35 系列镜头。手持拍摄，镜头全程保持轻微的浮动。
  • 主题与情绪：孤胆英雄、末日生存、机械与异兽的对抗、肃杀、气氛从紧张到狂热。
  • 声音：不要配乐，仅保留同期声。

关键元素

  • 主角：参考【图片】。
  • 武器：参考【图片】。
  • 敌人：参考【图片】。
  • 场景：参考【图片】。

画面内容

【氛围渲染】

  • 景别：极近距离特写。侧面 30° 拍摄主角在头盔内的面部特写。
  • 画面：人物轻微皱眉，眼神冷酷专注，仇视着前方。身体带有轻微的呼吸浮动。头盔玻璃亮起橙红色的数字 HUD，光影映射在她的眼睛和皮肤上。

【机械变形】

  • 景别：背部特写、手臂特写。
  • 画面：人物双手自然下垂，背部装甲与手臂装甲（精密的机械结构灵敏且迅速）层层折叠展开，内部暴露出散发着高温橙色光芒的机械核心。开启战斗模式，保持持续高亮。

【武器生成与充能】

  • 景别：手背特写平移至刀刃特写。
  • 画面：一只黑色的机械手掌活动手指，手掌位置生成一个机械化 3D 全息投影的科幻刀柄轮廓（轮廓：白色设计图线条）。轮廓逐渐长出刀身，直至完整设计线稿显现，科幻感十足。随后机械手掌握住刀柄，武器被激活，由白色线稿变为唐横刀实体，刀背带有科幻纹路。橙色能量由刀柄向刀刃充能，刀刃逐渐亮起。

【战斗准备】

  • 景别：人物全景。
  • 画面：人物姿态慢慢转变。武器充能完毕，暴露在外机械核心闭合，装甲回到初始状态。身体重心下沉，单手握刀，摆出经典的持刀战斗姿势。刀体散发的橙色光芒映衬着头盔和金属战衣，与周围的环境光形成冷暖对比。

【场景交代】

  • 景别：大全景。拍摄人物背面以及整个环境。
  • 细节：人物身处桥上，面对闸门严阵以待。远处的巨大闸门打开，门内是迷雾重重的废弃工厂。浓浓的雾气中出现多只身形高大的怪物狂奔，咆哮着朝人物方向袭来。

打斗（这个建议做 15s）

视觉基调

（同武器充能段，外加）：

  • 核心特效：热能武器高速挥动的过程中，光刃高亮区域更具移动轨迹留下橙色光拖尾。怪物被武器攻击部位飞溅出浓稠的蓝色光液，光液在 0.5 秒后黯淡熄灭。
  • 打斗动作：动作符合真实物理定律，节奏紧凑，拒绝游戏 CG 感。人物全程动作干脆利落、行云流水，拥有极致的爆发力与敏捷度，每一次攻击都如"外科手术"般精准攻击敌人要害，身手敏捷使敌人碰不到她。
  • 声音：不要配乐，仅保留同期声。

画面内容

【场景交代】

  • 镜头：大全景。
  • 画面：远处的巨大闸门打开，门内是迷雾重重的废弃工厂。浓浓的雾气中出现多只身形高大的怪物狂奔，咆哮着朝人物方向袭来。

【战斗爆发】

  • 镜头：动态运镜、慢动作混合快动作。
  • 画面：人物身处桥上，手拿武器面对闸门严阵以待。一只巨大的黑影怪物突然扑来。主角一个极具动感的侧身滑闪避随后立即起身。镜头追踪主角，她挥舞发光的唐横刀，在空中留下一道完美的橙色光轨，挥砍怪物。

【高速连斩】

  • 镜头（电影级运动跟随）：人物中景、脚部特写、头部特写、高空垂直俯拍全景。
  • 细节：主角目光带有仇恨，表情坚毅。在多只怪物间穿梭攻击，行云流水。将每一次攻击的剩余动能都转化为下一次攻击的起势。超高的机动性躲闪攻击。热能刀挥舞产生绚丽的橙色光能和粒子拖尾。每次击中怪物都会爆发出大量的蓝色的光液。

【终结一击 (慢动作)】

  • 镜头：极慢动作、强烈对比。
  • 画面：人物将刀精准划过一只怪物的身体，怪物的青蓝色眼睛在黑暗中极为醒目。粘稠如血液的蓝色光液溅出，在空中悬停。极具视觉观赏性。

【短暂喘息】

  • 镜头：拍摄人物背面。
  • 画面：主角背对镜头，帅气地在手中转动热能刀卸力，休息片刻。随后听到身后传来声响，气氛再次紧张，双手握住刀柄慢慢转身，带有轻微喘息，看向通道尽头的工业闸门。

【终极 Boss 登场】

  • 镜头：远景。
  • 画面：在充满雾气的工业通道尽头，一只体型更加巨大的怪物从工业闸门的雾中缓缓浮现。沉重的脚步声使得画面也有轻微震动。

【最终准备】

  • 镜头：刀刃特写、面部特写、人物全景。
  • 画面：热能刀在此充能，能量来到最大值。橙色的光刃亮度提高，映衬在人物的面部与装甲上。大战一触即发，人物摆出终极对决的姿态迎接强大敌人的挑战。

学习要点

  1. 武器充能 + 打斗 = 两段独立生成 + 后期剪辑。Mx-Shell 反复强调："一次出成品的概率极低"。
  2. 打斗段必须明确"动作符合真实物理定律"：AI 默认会做游戏 CG 感打斗，这一行是关键防御。
  3. "冷暖对比"是赛博朋克美学的灵魂：橙色（武器/装甲灯光/HUD）vs 青蓝色（环境/雾/怪物眼睛）。永远成对出现。
  4. 每个镜头都明确景别 + 镜头节奏：极慢动作、动态运镜、固定机位等，让 AI 知道这是个剪辑段不是一镜到底。

5 段式骨架

1. 核心主题      3-6 个风格标签，| 分隔。镜头类型 → 题材 → 美学参考。
2. 人物与场景    脸 / 服装 / 场景。「不美化」。先写材质。背景要有动态。
3. 氛围与质感    真实摄影机 + 镜头型号。色调。风格核心。声音行。
4. 镜头规则      单镜 / 角度（景别+角度+运动）/ 呼吸行。
5. 分镜          逐秒切片（一镜到底）或 逐镜切片（多镜剪辑）。
  • A 式 · 逐秒（变身、蓄力、一镜到底）：每片 动作 + 镜头 + 特效。可选：声音、表情。
  • B 式 · 逐镜（叙事、MV）：每镜 景别 + 构图 + 运镜 + 动作。

3 条反直觉原则

  1. 图片质量不高就别喂参考图。 AI 会模仿参考图的画风（CG感/二次元感），而不是设计。用文字描述放飞。例外：照片级 3D 渲染 / 真实高清实拍。
  2. 描述缺陷 = 描述真实。 太完美 = 假。满身战损 / 掉漆 / 关节有油污 / 保留面部细小瑕疵。每个人物/装备段落至少 2 处瑕疵。
  3. 抽卡是常态 —— 结尾留白。 每个镜头 2-3 到 20+ 次重抽很正常，别指望一次出片。结尾别堆特效。

两行原样粘贴

呼吸行（几乎每条都加 —— 两个限定词缺一不可，否则 AI 会理解成剧烈抖动）：

手持镜头。全程保持一种极其微弱、类似呼吸般的镜头浮动，以增强临场感。

声音行（否则 AI 会自己编配乐）：

声音：无配乐，仅保留现场同期声。

有标志性环境音的场景，要逐项列出（雨、雷、金属刮擦、低频能量嗡鸣），别让 AI 猜。

留白结尾模板

不要写：刺眼强光 / 爆炸 / 胜利姿势 / 跃向天空 / 镜头炸白。克制比堆砌更高级。

无对白。无爆炸。无强光。只有{{主体}}{{动作}}，{{环境细节}}。

示例：「只有一个穿着未完成战甲的身影站在原地。风卷起战场硝烟。一颗流星划过远空。」

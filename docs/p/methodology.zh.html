Mx-Shell 提示词方法论

这份文档把 Mx-Shell 在《丧尸清道夫》《假面骑士战衣》系列里反复使用的写法拆开讲。 不是模板复制说明书 —— 是让你自己能写出同等质量提示词的思路。

一句话总结

像写分镜脚本一样写提示词，而不是像写文学描述。 把场景按时间切片，每一秒钟分别交代「动作 / 镜头 / 特效 / 声音」这四件事。

5 段式结构

所有 Mx-Shell 的视频提示词都遵循这 5 段。顺序不要乱：

1. 核心主题       ← 给 AI 定调子（一行 tag）
2. 人物与基础设定 ← Who & What is wearing
3. 氛围与画质     ← How it looks
4. 运镜规则       ← How the camera moves
5. 分镜（时间轴） ← 按秒/按镜头展开剧情

段 1 · 核心主题

一行风格标签，用 | 或 丨 分隔。给 AI 一个总体审美锚点。

写法：3-6 个标签，从"画面类型"到"题材"到"美学风格"层层递进。

核心主题：写实暗黑特摄 | BLACK SUN美学 | 破碎肉身 | 战损变身 | 末日战场
核心主题：原子朋克 | 末日丧尸 | 电影级质感 | 超写实 | 杜绝游戏 CG 感
核心主题：写实科技感 | 科幻机甲 | 磅礴史诗 | 重工业机械美学 | 真人演绎

关键词清单（凑标签时可以挑）：

类型例
真实感锚点电影级质感、超写实、真人实景拍摄、杜绝游戏 CG 感、变形宽荧幕电影质感
美学定调暗黑写实、原子朋克、蒸汽朋克、赛博朋克、废土工业、复古港风
题材特摄剧、末日生存、孤胆英雄、机械武侠、重型机甲
情绪破碎肉身、战损变身、肃杀、磅礴史诗、压抑沉重

段 2 · 人物与基础设定

回答"画面里有谁、穿什么、站在哪"。三行：面部 / 服装 / 场景。

【人物与基础设定】
面部：参照上传图片，五官、脸型、发型百分百还原，杜绝美化。
      面部伤口，纱布，血渍保持一致。刘海覆额，表情全程阴郁。
      变身瞬间，眉心仅轻微蹙起，拒绝昂扬斗志与眼神高光，保持压抑感。
服装：哑光黑色皮质长风衣，金属腰带为哑光质地，腰带核心为暗红色水晶。
场景：末日战场，微风，硝烟弥漫，阴天荒野，灰蓝色调天空。
      天空中划过带有火焰与浓烟的陨石。

几个反直觉但 Mx-Shell 反复强调的要点：

  • "杜绝美化" — 真实感来自瑕疵。明确写"保留轻微面部瑕疵"比模糊地说"高质量"管用得多。
  • "参照上传图片，百分百还原" — 如果用了参考图，就强制锁定，防 AI 自由发挥。
  • 服装写质地 —"哑光黑色皮质"比"黑色皮衣"信息量大三倍。
  • 场景写动态环境 — 微风、硝烟、陨石。静态背景出不了氛围。

段 3 · 氛围与画质

模拟"真实摄影机 + 镜头"的写法。这一段是 Mx-Shell 最爱用的小技巧。

【氛围与画质】
视觉基调：变形宽荧幕电影质感。模拟 IMAX 胶片摄影机，
        搭配 Panavision C 系列镜头（焦段 35mm，光圈 f4）。
色彩与影调：低饱和灰蓝主调。暗部信息压缩，保留细节。
          边缘添加轻微柔焦与适度的胶片颗粒感。
风格核心：参考《XXX》美学风格。强调生物质感与外星科技的融合，
        营造压抑、沉重、实拍的肉身痛苦感。

为什么有效：AI 训练数据里大量电影摄影描述都和真实摄影机型号绑定。给 AI 一个具体摄影机型号 = 给它一个明确的视觉风格锚点。

Mx-Shell 常用的摄影机组合：

风格摄影机 + 镜头
史诗感 / 大场面IMAX 胶片摄影机 + Panavision C 系列镜头（35mm，f4）
暗调赛博 / 写实硬核索尼威尼斯电影机 + 佳能 K-35 系列镜头
港片 / 武侠柯达 35mm 复古胶片，跳过漂白胶片质感
商业人像Canon EF 85mm f/1.2

色调常用词：低饱和灰蓝 / 好莱坞青橙色调 / 60 年代复古暖橙 + 海盐蓝 / 暗调低照明高对比度。

段 4 · 运镜规则

定义镜头怎么动。三行：单镜头 / 角度 / 呼吸感。

【运镜规则】
单镜头：一镜到底，无剪辑。
角度：开场仰拍，人物半身左侧面 30 度，聚焦腰部以上部位。
     开始变身镜头以极缓速度环绕至正面平视上半身。
     变身完成后环绕至人物右侧面俯拍。
呼吸感：手持拍摄，全程保持极其轻微的、如呼吸般的镜头浮动，增强临场感。

关键概念词（描述镜头运动时往里挑）：

  • 景别：极近距离特写 / 特写 / 近景 / 中景 / 全景 / 大全景
  • 角度：仰拍 / 俯拍 / 平视 / 过肩镜头 / 高空垂直俯拍
  • 运动：极缓向前推进 / 环绕 / 跟随运动拍摄 / 固定机位 / FPV 模式 / 镜头后拉+下降+上摇
  • 节奏：极缓 / 匀速 / 突然 / 应激性 0.1 秒微颤 / 极慢动作

"呼吸感"这一行 —— Mx-Shell 几乎每个视频都写。原因：让 AI 加入轻微抖动，避免出 CG 感僵直镜头。

段 5 · 分镜（时间轴）

整套方法论里最关键、也最容易被忽略的部分。

两种写法，按场景选：

写法 A：按秒切片（适合单镜头一镜到底，比如变身、武器充能）

0-3秒 · 凝视
动作：主角略微低头，目光锁定腰带，脊背紧绷。右手缓慢抬起，握住腰带主体。
镜头：极缓向前推进，捕捉身体因呼吸产生的微幅起伏。
特效：整颗眼球骤然亮起白金色光芒，眼眶周围皲裂。
     眼睛亮起的炫光使画面镜头产生光斑。

3-6秒 · 启动
声音：低语：「HENSHIN」。
动作：手掌用力按压腰带核心，水晶核心本体被挤压而开裂，红光伴随晶体的裂纹亮起。
特效：金属机械装置被暴力唤醒，核心主体沿中线开裂。空气向内塌陷，产生畸变...
镜头：低频嗡鸣由远及近，镜头随之一同产生应激性的 0.1 秒微颤。

6-9秒 · 撕裂
[再按需细分：核心 / 身体 / 衣物 / 甲壳 / 面部 / 镜头]

每一段三件套：动作 + 镜头 + 特效。再加可选的「声音」「面部」等子项。

写法 B：按镜头切片（适合多镜头剪辑，比如《丧尸清道夫》）

分镜一：
景别：中景，拍摄机器人腿部以上位置。
构图：机器人背影作为前景居中位于画面中心，背景左侧落地窗，右侧吧台。
运镜手法：跟随运动拍摄。
画面内容：机器人背对镜头向前方移动，手舞足蹈进入别墅休闲区，
         潇洒转身作为舞蹈 END 动作，led 显示器保持绿色微笑表情。
         片刻后画外传来玻璃酒瓶被打碎的声音，机器人受到惊吓，
         下意识向画面左侧后撤半步，led 显示器瞬间切换至黄色害怕表情。

分镜二：
景别：全景，平视拍摄吧台全貌。
构图：空镜平视拍摄吧台全貌，画面下方三分之一处为吧台和吧台椅子，
     上方为墙面酒柜。
运镜手法：镜头缓慢前推，渲染氛围。
画面内容：吧台后方传来悉悉簌簌的声音，像是有什么东西躲在吧台下翻找。

每个分镜四件套：景别 + 构图 + 运镜手法 + 画面内容。

三个反直觉但有效的原则

1. 不要给 AI 参考图，除非画质够高

"我的所有假面骑士的战衣造型都是靠描述让 AI 自由发挥的。"

垫图最大的副作用：AI 容易复现参考图的画风（包括 CG 感、动漫感），而不是参考它的设计。除非参考图本身就是 3D 渲染或高画质实拍，否则纯文案描述效果更好。

2. 描述瑕疵 = 描述真实

"过于完美，就假。适当地保留缺陷不是坏事。"

战损痕迹触目惊心、各处磨损掉漆、关节部位的油污、保留轻微面部瑕疵、战衣整体远非平整 —— 这些"缺陷"是 Mx-Shell 作品质感的核心来源。

3. 抽卡是常态，结尾要留空白

200+ 个镜头，每个少则两三次、多则 20+ 次才能挑出能用的。不要追求一次成型。

结尾也别堆特效：

"没有台词。没有爆炸。没有光芒万丈。只有身穿不完整战衣的人站在原地。"

留白比堆砌更显高级感。

风格画质骨架（可复用片段库）

这是 Mx-Shell 在不同作品里反复使用的"氛围与画质"段落骨架，按题材分类。 用法：直接复制对应风格段落，粘贴到你的提示词里。

1. 暗黑特摄 / 战损美学

视觉基调：变形宽荧幕电影质感。模拟 IMAX 胶片摄影机，
        搭配 Panavision C 系列镜头（焦段 35mm，光圈 f4）。
色彩与影调：低饱和灰蓝主调。暗部信息压缩，保留细节。
          边缘添加轻微柔焦与适度的胶片颗粒感。
风格核心：写实暗黑战损美学。强调生物质感与外星科技的融合，
        营造压抑、沉重、实拍的肉身痛苦感。

适合：变身、机甲、末日英雄。

2. 好莱坞青橙 / 硬核赛博

画面风格：赛博朋克、硬核科幻、电影级质感、超写实、真人实景拍摄、废土工业风。
色彩与色调：好莱坞青橙色调。冷峻的橙灰色和暗青色，
          营造出压抑、冰冷和未知的氛围。高亮度的霓虹橙色和青蓝色。
          极端的"冷暖对比"为核心视觉特征。
光影设计：暗调低照明、高对比度。暗部信息压缩，保留细节。
        边缘添加轻微柔焦与适度的胶片颗粒感。画面整体可见度极低。
环境光：浓密的体积雾漫射出的微弱冷光。
光源：发光道具作为动态光源，产生强烈的边缘光和镜头光晕。
拍摄设备：索尼威尼斯电影机拍摄，搭配佳能 K-35 系列镜头。
        手持拍摄，镜头全程保持轻微的浮动。

适合：赛博朋克、动作打斗、机械与异兽对抗。

3. 60 年代原子朋克 / 复古未来

风格核心：原子朋克、电影级质感、超写实、极致逼真、真人实景拍摄、杜绝游戏 CG 感。
视觉基调：变形宽荧幕电影质感。使用 IMAX 胶片摄影机搭配 Panavision C 系列镜头
        （添加动态模糊）拍摄。
色彩与影调：60 年代复古科幻原子朋克美学，复古暖橙 + 海盐蓝高对比色调，
          胶片颗粒质感，复古广角镜头，低饱和度复古胶片滤镜，
          采用自然日光照明，大面积落地窗透进柔和的自然侧光，
          在水磨石地面形成自然柔和的光影。整体光线通透均匀，
          高光不过曝，暗部保留完整细节，明暗过渡自然，
          营造寂静的光影氛围，带轻微复古胶片柔光质感。
对标参考：参考美剧《辐射》美学风格、皮克斯动画人物动作控制。

适合：丧尸末日 + 度假 + 反差美学。

4. 邵氏港风 / 武侠机械

邵氏电影风格，蒸汽朋克与古风武侠的完美结合，
80 年代的技术彩色武术电影风格，复古港风，机械武侠，史诗构图，
深色阴影，柯达 35mm 复古胶片拍摄，跳过漂白胶片质感，
画面高光部位添加柔焦效果，画面质感丰富，
真人实景拍摄，写实电影感。

色彩与光影：非自然戏剧化人造光。主色调为冷色调，
          以青灰、银白和暗红色为主，烛火发出的暖光映衬着面部。
          光线是带有电影感的硬光，
          在面部和金属上形成了清晰的明暗对比。

适合：东方古风 + 蒸汽朋克 / 武侠 + 机械。

5. 重型机甲 / 史诗工业

模拟设备：模拟 IMAX 胶片摄影机，搭配 Panavision C 系列镜头。
        帧率 24，模拟动态模糊。
色彩与影调：好莱坞青橙色调，低饱和度。暗部信息压缩，保留细节。
          边缘添加轻微柔焦与适度的胶片颗粒感。
风格核心：参考重型机甲电影美学。画风写实，科幻史诗，
        重型机甲，重工业机械美学与先进科技完美融合，
        保持原始的机械感与重量感。
核心特效：好莱坞级视觉特效。

适合：人形机甲 / 机械变形 / 大场面对决。

6. 商业人像（用于做参考图）

图片风格：「人像摄影」。生成 22-25 岁年轻 {{亚裔/欧美/...}} {{男性/女性}}
        的超写实商业人像摄影，近景。
特征：五官、脸型与参考照片保持 100% 一致；保留轻微面部瑕疵；
     带有少量自然碎发以营造慵懒松弛的氛围；无任何配饰。
服装：{{深黑色 T 恤 / ...}}。
背景：浅白色的模糊渐变质感背景。
光线：柔和的侧光，突出面部的立体感。
皮肤：暖调健康肤色并带有自然细腻的光泽感，
     细节增强度调至 100%，关闭过度磨皮功能。
色彩风格：高级柔和的暖色调，边缘锐化清晰且无模糊、锯齿或重影。
镜头：Canon EF85mm f/1.2。
比例：3:4。

适合：把照片转成「商业感人像」再用于视频生成的中间步骤。绕过 Seedance 人脸审核的常用方法。

怎么混搭

这些骨架可以拼接。例：

  • 暗黑特摄 + 工业机甲 = 战损变身后场景拉远 + 机甲俯瞰末日战场
  • 原子朋克 + 邵氏港风 = 60 年代怀旧色调 + 东方武侠人物
  • 好莱坞青橙 + 重型机甲 = 海上夜战 + 机甲砸海

关键：色彩与影调段尽量统一，否则前后镜头剪辑时会色差严重。
